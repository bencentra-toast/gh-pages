Certain columns in this table are shown are hidden based on screen size:

  • Date column is always present
  • Upload ID column is only present on Desktop (> 1024px)
  • Created, Attached, and Updated columns are present on Tablet and Desktop (> 768px)
  • Operations column is only present on Mobile (< 768px)

In this example, the desktop and mobile states are both in the DOM simultaneously. Will this cause issue with accessibility (screen readers, etc)?
